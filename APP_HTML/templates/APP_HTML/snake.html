Snake Game

Just a simple snake game created with JavaScript. Logically, there are many ways to tackle a game like snake. In this situation, I thought it would be best to create the snakes environment using a grid of rows and columns. This gives the ability to quickly scale the map size by changing the number of rows and columns -- which, in turn allows the use of percentages within css. This also resolves any alignment issues.

The objective is simple; use the keyboard arrow keys to collect as much food as possible without hitting walls or yourself. The score can be seen in the bottom left (but it is not recorded).
