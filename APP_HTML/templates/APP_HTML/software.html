Websites:

The most recent website I created is this one! I have also created and worked on at least 30 others. The reason I chose to create this project using Django was to teach myself something new while also being productive. I am very familiar with: Python, HTML, CSS, JS and knew this project would be an enjoyable way to display my skills.
One of the things I gave high priority to in this project was scalability. After all, scalability is one of Django's greatest benefits. Something I quickly learned was the ability to reduce redundant code while increasing efficiency.

For example: the navigation bar is on every webpage. Instead of pasting the same code at the top of each html page I created one html page for the navigation bar alone. This not only makes the code easier to navigate, but whenever changes need to be made one page is edited opposed to editing all of them. The result of fully applying this logic can be seen below.

Below is an example using the resume page:

  1. <!DOCTYPE html>
  2. {% load static %}
  3. <html>
  4.   <head>
  5.     {% include 'FILE/head.html' %}
  6.     <title>Resume</title>
  7.   </head>
  8.   <body class="defaultPG">
  9.     <iframe class="noScrollPG" style="top:5%;" src="{% static 'FILE/resume.pdf' %}"></iframe>
  10.     {% include 'FILE/footer.html' %}
  11.   </body>
  12. </html>
